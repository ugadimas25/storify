## TECH REQUIREMENTS

- **Framework**: React with TypeScript (or plain React if simpler)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion for scroll-triggered animations (fade-in, slide-up)
- **Icons**: Lucide React
- **Sections**: Each section as a separate component
- **Responsive**: Fully mobile-responsive (mobile-first)
- **Performance**: Lazy load images, smooth scroll behavior
- **State**: Minimal — only for mobile menu open/close and FAQ accordion

---

## DESIGN DETAILS

- Use `backdrop-blur` glass cards for feature cards
- Hero section: animated gradient mesh or particle effect in background
- Section transitions: alternating dark (#0F0F1A) and slightly lighter (#13131F) backgrounds
- Cards: subtle border (`border border-white/10`), hover lift effect (`hover:-translate-y-1`)
- All CTAs use amber/orange gradient: `from-amber-500 to-orange-500`
- Section headings: gradient text (purple to violet)
- All text: white/gray scale on dark backgrounds
- Border radius: rounded-2xl for cards, rounded-full for badges and buttons

---

## COPY STYLE

- Language: Bahasa Indonesia
- Tone: Warm, inspiring, confident — like a friendly brand that cares
- Avoid jargon — speak like a friend recommending a product
- Use "kamu" (informal) for friendly tone, not "Anda"
- Highlight benefits over features — "dengarkan kapan saja" not "background audio playback"
